# Howling MUNE Client Flow Engine

Next.js, TypeScript, Tailwind CSS, and shadcn-compatible landing page foundation for Howling MUNE.

## Status

Initialized from an empty repository with:

- App Router structure
- TypeScript config
- Tailwind config
- global style tokens
- `/components/ui/minimalist-hero.tsx`
- `/lib/utils.ts`
- `/app/page.tsx`
- `/DESIGN.md`
- `/reference/howling-mune-reference.html`

## Install

```bash
npm install
```

## Run

```bash
npm run dev
```

## UI path

Reusable UI components live in:

```txt
/components/ui
```

This keeps the project aligned with normal shadcn conventions and makes future component work predictable.

## Main design direction

Quiet lunar premium. Dark tactile surfaces. Amber/gold signal. Human pathway language. No generic wellness template. No fake metrics.
